# Apple Store Clone

A responsive frontend clone of the Apple Store homepage built with HTML, CSS, and JavaScript.

## Project structure

- `index.html` — main homepage
- `assets/css/styles.css` — modern responsive styling
- `assets/js/main.js` — navigation toggle and footer year script
- `assets/images/` — local image assets used in the layout

## Usage

1. Open `index.html` in a browser.
2. Edit `assets/css/styles.css` to customize colors, layout, and spacing.
3. Add or replace images in `assets/images/` if needed.

## Improvements included

- semantic HTML structure
- responsive navigation and layout
- improved homepage hero and category sections
- simplified asset organization for easier editing
